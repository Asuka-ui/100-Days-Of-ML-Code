我无意间发现了这个东西感觉也还很不错于是打算跟着这个项目进行学习，并把其中一些弃用的函数替换成自己的东西。<br>
顺便也在这个过程中学习使用markdown和git这些工具。<br>
项目中的注释是我自己的一些粗浅的想法，大佬轻喷。T……T
